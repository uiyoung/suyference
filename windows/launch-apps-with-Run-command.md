# Launch Apps With the Windows Key + R Shortcut

`C:\Users\{유저명}` 에 바로가기를 넣어두면 '실행'에서 실행 가능

ex)`unity.lnk` 바로가기를 `C:\Users\{유저명}`에 넣어두면 `Windows Key + R`-> `unity`로 실행 가능


## commands
- shutdown -s : shutdown
- shutdown -r : restart
- taskmgr : Task Manager
- control : Control Panel
- notepad
- mspaint
- calc
